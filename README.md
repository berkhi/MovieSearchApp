# MovieSearchApp
This application is the individual final project of the Pazarama iOS Bootcamp.

# Technologies and Libraries
* Protocol Oriented Programming
* URLSession
* Programmatic UI
* MVVM
* Delegate Pattern
* Singleton Pattern
* Unit Tests
* Kingfisher
